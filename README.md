# Final Laboratory Examination: Java Swing To-Do List Application

## Overview
This project is a Java Swing To-Do List application that allows users to add,
view, and manage tasks using a structured object-oriented approach. The program
uses a table-based interface, a separate task input form, and a TaskManager
controller for task operations.

## Features
- Display task list using JTable with Task ID, Name, Description, and Status
- Add new tasks through a separate form window
- Ensure only one instance of the form window is open at a time
- Validate input before saving tasks
- Use TaskManager to handle task operations and data management
- Refresh the task table automatically after adding new tasks
